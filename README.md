<h2 id="objective">Objective</h2>

My main goal in doing this project was mainly to learn how to use the pygame library and develop my python skills. When choosing to create the famous snake game, I decided that I would add something more to this challenge, so I decided to put obstacles that appear every 2 apples eaten, giving an extra challenge to the players and also to me, who was programming.

![demo](https://github.com/MuMartins/Snake-Game/tree/main/ASSETS/demo.gif?raw=true)

<h2 id="features">Project features</h2>

- [x] The classic snake game
- [x] Creating obstacles
- [ ] Snake with different skin

<h2 id="prerequisite">Prerequisite</h2>

Before starting, you will need to have the following tools installed on your machine:
- To run it you will need [Git](https://git-scm.com) and the newest version of [Python](https://www.python.org/downloads/)
- And of course, you need to install the [Pygame](https://www.pygame.org/) library

<h2 id="pygame">Installing Pyagme</h2>

Make sure to check the pip option while installing Python in yout machine. After that just open your cmd and install Pygame:

```bash
# Installing Pygame to run the game
$ pip install pygame
```
